# Requirements
pytesseract 모듈을 사용하여 OCR 작업을 수행하려면 Tesseract OCR 엔진을 설치해야 함.  pytesseract는 이 엔진을 python에서 사용할 수 있도록 인터페이스를 제공
* Tesseract OCR 엔진 설치
mac일 경우 brew, 리눅스일 경우 sudo
```
brew install tesseract
sudo apt install tesseract
```
* pytesseract 설치
```
pip install pytesseract
```
